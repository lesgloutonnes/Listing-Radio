CHC - Listing Radio
===================

Module Odoo de listing et gestion équipement d'imagerie médicale.

Installation
------------

1. Copiez le dossier `chc_radio_listing` dans vos addons Odoo.
2. Mettez à jour la liste des applications.
3. Installez ou mettez à jour le module `CHC - Listing Radio`.
4. Assignez les droits aux utilisateurs :
   - Utilisateur : consultation.
   - Gestionnaire : création, modification, suppression, import et correction.
5. Ouvrez le menu `CHC - Listing Radio > Listing Radio`.

Les données sont stockées dans la base PostgreSQL Odoo.
